Our program should be run according to all of the directions laid out in the demo that we were provided.

Andrew Camps: Mainly worked on creating queries on the database
Garrett Seale: Mainly worked on adding, updating and deleting from database items from the database
We both worked on front and backend

Main Page - From here all pages can be reached. Using the drop down menues or link a user can add, update and delete items from the database as well as make queries.

Add Student
- Fill out the form in full. All fields are required and the form will only submit if all fields are filled out. A message will pop up saying a student was successfully saved.

Add Staff
- Use the drop down menu to select staff to add. Simply fill out the form and wait for alert to say staff member was accepted.

Add Lease
- To create a lease, you just pick the student and room you want to put on the lease. A space to type in the name is provided and a couple of links to check vacant rooms as well as the current rent of rooms. Rent can be typed in at any rate.

Update Rent
- Simply add the new rent value to the room and click save
- The page will reload displaying the new value
- This will also update the rent of all future leases

Delete Student
- Simply click delete and the page will reload without the deleted student

Resident Hall Info
- Displays the results of query #1

Lease Info
- Displays the results of query #2

Unpaid Invoices
-Displays results of query #3

Vacant Rooms
- Satisfies requirements of one of the group defined queries
- Displays a list of vacant rooms

Room Inspections
- Satisfies requirement of query that takes user input
- Lists all inpections that are related to a room
